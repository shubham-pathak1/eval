# eval

a minimalistic glassmorphic calculator for quick calculations.

## Usage

- **Toggle**: `Alt + Space`
- **Clear/Hide**: `Escape`

Matches live as you type.

## Development

```bash
# install
npm install

# dev
npm run tauri dev

# build
npm run tauri build
```

## Tech

- Tauri 2.0
- React + TypeScript
- mathjs
